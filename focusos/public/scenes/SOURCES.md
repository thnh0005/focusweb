# Scene Image Sources

The current focus room scene images are user-provided generated assets from July 10, 2026 and converted to WebP for the app.

Files:

- `rainy-cafe.webp`: rainy study window scene.
- `forest-cabin.webp`: forest cabin with rain and fireplace.
- `night-study.webp`: city night study room.
- `hobbit-library.webp`: warm round-window library.
- `train-window.webp`: sunset train study scene.
- `da-lat-mist.webp`: mountain morning room scene.

Keep replacement scene assets in this folder and update `src/constants/focus-scenes.ts` if file names change.
